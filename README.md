Hospital Management Dashboard (Excel)
📌 Project Overview

This project is an interactive Hospital Management Dashboard built using Microsoft Excel. It helps analyze hospital operations by tracking patient admissions, doctor availability, billing, and overall performance through dynamic dashboards and KPIs.

🎯 Objectives
Analyze hospital data efficiently.
Monitor patient admissions and discharge trends.
Track department-wise performance.
Visualize key hospital metrics.
Support data-driven decision-making.
🛠️ Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Excel Functions (IF, SUMIFS, COUNTIFS, XLOOKUP/VLOOKUP)
📊 Dashboard Features
Total Patients
Total Revenue
Patient Admission Analysis
Department-wise Patient Count
Doctor Availability
Billing Summary
Interactive Filters using Slicers
📈 Key Insights
Identified patient admission trends.
Compared department-wise performance.
Analyzed revenue generated from hospital services.
Improved data visualization using interactive dashboards.
